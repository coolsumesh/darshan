# Darshan Web UI (Prototype)

Static HTML prototype for a **read-only Profile Viewer**.

## Run

Open `web/index.html` in a browser.

If you want a local server:

```bash
cd web
python3 -m http.server 8080
# then visit http://localhost:8080
```

## Notes

- **Prototype / mocked:** profile contents are hardcoded strings.
- Includes a warning banner: "Profiles may contain sensitive info. Access is audited." (future backend/auditing).
